# 🏢 Employee Attrition Prediction App

A Machine Learning web application built using Streamlit that predicts whether an employee is likely to leave the company or stay based on HR-related features.

## 📊 Project Overview

Employee attrition is a major challenge for organizations. This app uses a trained ML model to analyze employee data and predict whether an employee will stay or leave the company.

It helps HR teams:
- Identify high-risk employees
- Take preventive actions
- Improve employee retention

---

## 🧠 Machine Learning Model

- Type: Classification Model
- Algorithm: Logistic Regression / Random Forest (based on training)
- Dataset: HR Employee dataset
- Features: 34 employee attributes
